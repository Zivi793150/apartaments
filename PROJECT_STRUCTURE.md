📊 ПОТОК ДАННЫХ

Пользователь → Браузер → React компонент → Mapbox GL JS
                              ↓
                        useState hooks
                              ↓
                        GeoJSON слой
                              ↓
                        3D визуализация
                              ↓
                        WebGL Canvas
                              ↓
                        Экран пользователя

1. Получить токен: https://www.mapbox.com/
2. Добавить в .env.local
3. npm run dev
4. Открыть http://localhost:3000/spain-building
