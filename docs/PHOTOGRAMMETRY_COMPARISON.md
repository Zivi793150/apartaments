# Сравнение инструментов фотограмметрии

## Краткое сравнение

| Инструмент | Платформа | Стоимость | Сложность | Качество | Время |
|------------|-----------|-----------|-----------|----------|-------|
| **RealityScan** | iOS/Android | Бесплатно | Низкая | Хорошее | 1-2 часа |
| **Meshroom** | Desktop | Бесплатно | Средняя | Отличное | 2-4 часа |
| **RealityCapture** | Desktop | Платно | Средняя | Отличное | 1-3 часа |
| **3DF Zephyr** | Windows | Бесплатно/Платно | Средняя | Хорошее | 2-3 часа |
| **COLMAP** | Desktop | Бесплатно | Высокая | Отличное | 3-5 часов |

---

## RealityScan ⭐ (Рекомендуется для заказчика)

### Преимущества:
- ✅ **Бесплатно** - нет ограничений
- ✅ **Мобильное приложение** - можно использовать на месте
- ✅ **Простой интерфейс** - заказчик может сам использовать
- ✅ **Быстрая обработка** - 20-40 минут
- ✅ **Хорошее качество** - достаточно для веб-приложения
- ✅ **Экспорт в GLB** - готово к использованию

### Недостатки:
- ⚠️ Требует хорошее освещение
- ⚠️ Нужно много фотографий (30-50+)
- ⚠️ Обработка на устройстве (может быть медленной)
- ⚠️ Ограниченные настройки оптимизации

### Когда использовать:
- Заказчик может сам сфотографировать
- Нужно быстро получить результат
- Не требуется максимальное качество
- Бюджет ограничен

**Вердикт:** Лучший вариант для заказчика, который может сам сфотографировать здание.

---

## Meshroom (Альтернатива для профессионалов)

### Преимущества:
- ✅ **Бесплатно** и open-source
- ✅ **Высокое качество** - профессиональные алгоритмы
- ✅ **Гибкие настройки** - полный контроль
- ✅ **Работает офлайн** - не требует интернет

### Недостатки:
- ⚠️ Только для Windows/Linux
- ⚠️ Требует мощный компьютер (GPU)
- ⚠️ Сложный интерфейс
- ⚠️ Долгая обработка (2-4 часа)

### Когда использовать:
- Нужно максимальное качество
- Есть опыт работы с 3D
- Есть мощный компьютер
- Есть время на обработку

**Вердикт:** Лучший бесплатный вариант для профессионалов.

---

## RealityCapture (Профессиональный вариант)

### Преимущества:
- ✅ **Лучшее качество** - промышленный стандарт
- ✅ **Быстрая обработка** - оптимизированные алгоритмы
- ✅ **Много функций** - полный набор инструментов
- ✅ **Хорошая документация** - подробные инструкции

### Недостатки:
- ⚠️ **Платно** - от $39/месяц
- ⚠️ Требует мощный компьютер
- ⚠️ Сложный интерфейс
- ⚠️ Кривая обучения

### Когда использовать:
- Профессиональный проект
- Нужно максимальное качество
- Есть бюджет
- Есть опыт работы

**Вердикт:** Лучший вариант для профессиональных проектов с бюджетом.

---

## 3DF Zephyr (Компромисс)

### Преимущества:
- ✅ **Есть бесплатная версия** - до 50 фотографий
- ✅ **Хорошее качество** - профессиональные алгоритмы
- ✅ **Простой интерфейс** - проще чем Meshroom
- ✅ **Быстрая обработка** - оптимизировано

### Недостатки:
- ⚠️ Только для Windows
- ⚠️ Бесплатная версия ограничена
- ⚠️ Платная версия дорогая ($149+)

### Когда использовать:
- Windows пользователь
- Нужно качество выше RealityScan
- Готовы платить или использовать бесплатную версию

**Вердикт:** Хороший компромисс между простотой и качеством.

---

## COLMAP (Для экспертов)

### Преимущества:
- ✅ **Бесплатно** и open-source
- ✅ **Максимальное качество** - научные алгоритмы
- ✅ **Полный контроль** - все настройки доступны
- ✅ **Активно развивается** - постоянные обновления

### Недостатки:
- ⚠️ **Очень сложный** - требует знания терминологии
- ⚠️ Командная строка - нет GUI
- ⚠️ Долгая обработка (3-5 часов)
- ⚠️ Требует опыт работы

### Когда использовать:
- Нужно максимальное качество
- Есть опыт работы с командной строкой
- Есть время на настройку
- Исследовательский проект

**Вердикт:** Только для экспертов, которые готовы потратить время.

## Рекомендации по выбору

### Для заказчика (самостоятельная съемка):
1. **RealityScan** ⭐ - лучший вариант
   - Простой
   - Бесплатный
   - Достаточное качество

### Для профессионала (высокое качество):
1. **RealityCapture** - если есть бюджет
2. **Meshroom** - если бюджет ограничен
3. **3DF Zephyr** - если Windows и нужен баланс

### Для быстрого результата:
1. **RealityScan** - самый быстрый
2. **Простая геометрия + фото** - если не нужна 3D модель
