### Шаг 1️⃣: Получить Mapbox токен
→ Перейти: https://www.mapbox.com/
→ Sign Up (бесплатно)
→ Account → Tokens → скопировать Public token (pk.eyJ...)

### Шаг 2️⃣: Добавить токен
Создайте файл: `c:\Users\zivi7\apartaments\.env.local`

Содержимое:
```
NEXT_PUBLIC_MAPBOX_TOKEN=pk.eyJ1Ijoi_YOUR_COPIED_TOKEN_HERE_
```

### Шаг 3️⃣: Запустить
```bash
cd c:\Users\zivi7\apartaments
npm run dev
```

Откройте: http://localhost:3000/spain-building

**✅ ГОТОВО!** Вы видите 3D визуализацию!

## 📍 ЕСЛИ ХОТИТЕ ДРУГОЙ АДРЕС

Добавьте в `.env.local`:

```env
NEXT_PUBLIC_BUILDING_LAT=40.4168     # Широта
NEXT_PUBLIC_BUILDING_LNG=-3.7038     # Долгота
```

Примеры:
- Мадрид: LAT=40.4168, LNG=-3.7038
- Барселона: LAT=41.3851, LNG=2.1734
- Нью-Йорк: LAT=40.7128, LNG=-74.0060

## ❓ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

**Карта не загружается?**
→ Проверьте токен в .env.local
→ Перезагрузите сервер (Ctrl+C, npm run dev)

**Квартиры не видны?**
→ Откройте F12 → Console → проверьте ошибки

**Слишком медленно?**
→ Обновите браузер (Chrome, Firefox, Safari, Edge)
